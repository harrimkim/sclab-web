Scalable Computing Lab.

저희 랩에서는 계산과학과 고성능 컴퓨팅에 대해서 연구합니다. 기계공학, 컴퓨터공학등에서 널리쓰이는 수치 최적화, 메쉬 생성, 메쉬 최적화, 메쉬 변형, 시뮬레이션 모델링 등에 대해서 연구하고, 최근에 각광받고 있는 슈퍼컴퓨터와 같은 멀티코어 환경에서 병렬 컴퓨팅 (OpenMP, MPI, CUDA)을 이용한 계산속도 향상, 계산 정확도 향상등에 대해서 연구합니다. 이외에도 요새 컴퓨터 공학 분야에서 각광 받는 기계학습, 패턴인식, 딥 러닝에 관해서도 연구합니다.
